# Self-Correction & Reflexion Loops Arena

A premium, full-stack application demonstrating state-of-the-art **Self-Correction and Reflexion agentic loops** built using **LangGraph**, **Google Gemini**, **Groq Llama 3.3**, and **Flask**.

## 🚀 Key Features

### 💻 Automated Code-Correction Subprocess Compiler
- **Sandbox Execution**: Code generated by the agent is executed in an isolated, secure Python subprocess sandbox.
- **Traceback Interception**: The sandbox catches any runtime/syntax stack traces (e.g. `NameError`, `IndexError`, `SyntaxError`) and standard console logs.
- **Feedback Reflexion**: If the script crashes, the exact traceback, failing code context, and stdout are compiled and fed back into the LLM node, which automatically isolates the bug and applies repairs iteratively!
- **Iteration Limits**: Standard 4 attempts guard limit to prevent runaway loops.

### 🔍 Corrective RAG (CRAG) & Self-RAG
- **Term Document Retrieval**: Retrieves context matches from a local mock collection.
- **Relevance Evaluator Node**: An objective grading node scores each document's query coverage, identifying information gaps (e.g., temporal changes).
- **Web Search Fallback**: Automatically triggers Web Search to patch context gaps.
- **Self-RAG Grounding Audit**: Grades whether the generated answer is grounded in facts or contains hallucinations. Loops back to regenerate if hallucination scores are low.

## 🛠️ Startup Instructions

1. Ensure [Astral `uv`](https://github.com/astral-sh/uv) is installed.
2. Verify credentials inside the local `.env` file.
3. Open a terminal inside the project folder:
   ```bash
   cd "d:\Agentic AI\langgraph-self-correction"
   uv run python app.py
   ```
4. Visit **`http://127.0.0.1:5001`** in your browser.
